====================================
STEP 1 — CREATE GITHUB ACCOUNT
====================================
Go to: https://github.com/signup
  Email: your email
  Username: bhuvilife (or any)
  Password: create strong one
  Verify email

====================================
STEP 2 — CREATE GITHUB REPOSITORY
====================================
After login go to: https://github.com/new
  Repository name: memora
  Description: MemorA - Your Personal AI Memory
  Public (free hosting needs public repo)
  Click: Create repository

====================================
STEP 3 — UPLOAD FILES TO GITHUB
====================================
On the repository page click: uploading an existing file
OR click: Add file -> Upload files

Upload these 4 files:
  memora_cloud.py
  memora_cloud_db.py
  requirements.txt
  render.yaml

Click: Commit changes

====================================
STEP 4 — CREATE RENDER ACCOUNT
====================================
Go to: https://render.com
Click: Get Started for Free
Sign up with GitHub (click Continue with GitHub)
Authorize Render

====================================
STEP 5 — DEPLOY ON RENDER
====================================
After login click: New +
Click: Web Service
Click: Connect a repository
Select: memora (your GitHub repo)
Click: Connect

Settings page:
  Name: memora
  Region: Singapore (closest to India)
  Branch: main
  Build Command: pip install -r requirements.txt
  Start Command: python memora_cloud.py
  Instance Type: Free

Click: Create Web Service

====================================
STEP 6 — WAIT FOR DEPLOY
====================================
Render will build and deploy automatically.
Takes 3-5 minutes.
Watch the logs — when you see:
  MemorA Cloud starting...
It is live.

Your URL will be:
  https://memora-xxxx.onrender.com
